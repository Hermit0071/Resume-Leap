# Resume-Leap
# 简跃 · 升学简历生成器

这是一个适合 macOS 使用的本地简历生成器，面向高中生申请活动、夏校、竞赛、营队或其他升学项目。

支持功能：

- 3 套预设风格模板
- 自定义基本信息与各类简历板块
- 8 类预设板块，可按需添加
- 新增、删除、隐藏、排序板块与条目
- 微调主题色、字体大小、排版密度
- 导出 `PDF`、`DOCX`、`PNG`
- 本地浏览器实时预览
- 独立“我的简历”页面与本地历史记录
- PDF、PNG 优先使用内嵌 Chrome 渲染器，失败时自动切换备用引擎

## 启动方式
直接双击根目录中的 `简历生成器.html`。

每次打开都会创建一份空白简历；编辑后的版本会自动保存在浏览器本地的“我的简历”板块。

PDF 导出会打开 macOS 打印窗口，请在窗口中选择“存储为 PDF”；DOCX 和 PNG 会直接下载。

## 推荐板块

你可以按需保留或删除这些内容：

- 教育背景
- 活动与领导力
- 竞赛与荣誉
- 志愿服务 / 项目经历
- 技能与语言
- 个人陈述亮点
